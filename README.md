# UnexpectedGardener

Живые хитрости для дома и огорода: хлорелла, вода, почва по Хольцеру.  
Без скучной теории, с юмором и граблями.

**Статус:** в работе  
**Лицензия:** Все права защищены (No license, пока не решена монетизация)

## О проекте

Это сборник неочевидных, проверенных на собственных граблях методов для:
- домашнего хозяйства (микрозелень, хлорелла)
- приусадебного участка (подход Зеппа Хольцера)
- работы с водой (pH, ORP, водородная вода)

Никакого назидания. Только «я попробовал, и вот что вышло».

## Структура

- `chapters/` — тексты глав
- `diagrams/` — схемы (от руки и фото)
- `tables/` — таблицы параметров, решений
- `experiments/` — журналы экспериментов
